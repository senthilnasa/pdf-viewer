# Contributing to PDF Viewer Platform

Thank you for considering a contribution!

## How to Contribute

1. **Fork** the repository on GitHub
2. **Clone** your fork locally
3. Create a **feature branch**: `git checkout -b feature/my-feature`
4. **Commit** your changes with clear messages
5. **Push** to your fork and open a **Pull Request**

## Code Standards

- PHP 8+ syntax only
- Use PDO prepared statements — no raw SQL interpolation
- CSRF protection on all POST forms
- Validate and sanitize all user input at the boundary
- Follow the existing file/class naming conventions

## Reporting Issues

Use GitHub Issues. Please include:
- PHP and MySQL versions
- Steps to reproduce
- Expected vs actual behaviour
- Relevant error messages or logs

## Pull Request Guidelines

- Keep PRs focused — one feature or fix per PR
- Update relevant documentation when adding features
- Test on PHP 8.0+ and MySQL 5.7+ before submitting
